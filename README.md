# PixelSmile for ComfyUI (Unofficial)

An unofficial implementation of [PixelSmile](https://github.com/Ammmob/PixelSmile) workflow in ComfyUI.

## Introduction

This project leverages AI to bring the PixelSmile expression control workflow into ComfyUI. With minimal modifications to the conditioning logic, it enables fine-grained expression editing—from neutral to specific expressions—using natural language prompts.
